# 📚 **Online Book Exchange Portal**

> A modern **MERN Stack Web Application** developed to help students exchange, buy, and share academic books within the campus community.

---

# 🚀 **Project Overview**

The **Online Book Exchange Portal** is a full stack web application where students can:

✅ Add Book Listings  
✅ Search Available Books  
✅ View Book Details  
✅ Delete Listings  
✅ Manage Book Exchange Easily  

This project was developed using **React, Node.js, Express.js, and MongoDB**.

---

# 🛠️ **Technologies Used**

## 🎨 Frontend

- **Axios** – API communication
- **React Icons** – Modern UI icons

---

## ⚙️ Backend

# 🔄 **Project Workflow**

```text
User Adds Book
       ↓
React Frontend
       ↓
Express Backend APIs
       ↓
MongoDB Database
       ↓
Books Displayed on Website

---

## ⚙️ Project Workflow

1. User enters book details in the form.
2. React frontend sends data to backend APIs.
3. Express server processes requests.
4. MongoDB stores the book information.
5. Stored books are displayed dynamically on the website.
